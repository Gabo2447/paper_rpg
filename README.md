# RPGPlugin - Minecraft Paper Plugin
A basic RPG plugin for Paper/Spigot version 26.2 that adds a system of active and passive skills with data persistence in YAML.

## Skills Included

- **Lava Leather (passive):** 25% chance of withstanding the fire and gaining strength
- **Wind Leap (Active):** Wind jump boost using vector physics, with a timestamp-based cooldown system and protection against elytra exploits.

## Technologies

- Java
- Spigot/Paper API
- Adventure API
- YAML
